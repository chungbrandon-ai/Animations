# Minimalist Mathematical Narrative Template
## The Skeleton Template
Task: Create a self-contained HTML/Tailwind/JS web application that functions as an interactive "Mathematical Narrative" about [TOPIC NAME].
1. Mathematical Core:
Implement a [NUMERICAL METHOD] for the [DIFFERENTIAL EQUATION OR ALGORITHM]. The simulation should plot in real-time on a responsive HTML5 Canvas.
2. Aesthetic & UI:
Theme: "Tech-Noir Archive." Use a deep charcoal background (#161616) and a high-contrast primary accent like [COLOR NAME] ([HEX CODE]).
Typography: Pair 'Roboto Mono' for data/labels with 'Lora' (italic) for narrative headings.
Layout: A centered, bordered canvas container with a sleek "Math Info" overlay in the corner.
3. Narrative Structure:
Build a "Slide System" using a narrative overlay that appears once the simulation finishes.
Include 5 chapters that weave the history of the math ([SCIENTISTS/DEVELOPERS]) into a philosophical discussion of [PHILOSOPHICAL THEME].
4. Interactive Finale:
The final chapter must trigger a symbolic visualization.
Create a [VISUALIZATION TITLE]: A [STABLE ELEMENT DESCRIPTION] (The [CONCEPT A]) overlaid with a [CHAOTIC ELEMENT DESCRIPTION] (The [CONCEPT B]), both animated to represent the interplay between [CORE DUALITY, e.g., Destiny and Free Will].
5. Requirements:
The code must be a single, production-ready HTML file. Use Tailwind CDN for styling. Ensure smooth CSS transitions between the canvas and the narrative overlay.
6. Must have buttons on the bottom of the screen to move between slides
7. on animation slides have a play pause speed up slow down and next and back buttons
8. ui margins must be outlined
9. all buttons must work.
10. Read the final line of this prompt in context with the skeleton template to understand the research subject
11. includes quotes and facts over spontaneous creative writing
